# Як створити SMS-розсилку в Gatum

1. Увійдіть у кабінет Gatum (https://app.gatum.io).
2. Перейдіть у розділ **Кампанії** → **Нова розсилка**.
3. Завантажте список номерів (CSV) або оберіть сегмент.
4. Вкажіть **Sender ID** та текст повідомлення (до 160 символів для GSM-7).
5. Оберіть **маршрут** (direct / HLR-verified / MNP-aware).
6. Натисніть **Запустити** або заплануйте час старту.

Після запуску статус кампанії зміниться на `running`, потім `completed`.
